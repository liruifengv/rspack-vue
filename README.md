# rspack-vue

<image src="https://images.sayhub.me/blog/rspack-vue/rspack.png" width="300" height="250">

An example of a vue project using [Rspack](https://github.com/web-infra-dev/rspack).

## Technology Stack

- Vue3
- Rspack
- vue-router
- pinia
- JSX

## development

```bash
# install
pnpm i 
# develop
pnpm run dev
# build
pnpm run build
```
